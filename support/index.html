HELP & CONTACT

UpTier Support

Report a problem through UpTier's private support mailbox without exposing financial information.

Contact support

Email uptier.support@gmail.com for product support, privacy questions, security reports, or data-deletion requests. For a security matter, use the subject UpTier security report. Send only a short description first and request a secure transfer method before providing sensitive technical evidence.

Email private supportView known issues

Never email or publicly attach bank statements, broker exports, account numbers, login details, access tokens, Plaid credentials, API keys, MFA secrets, recovery codes, or screenshots containing private financial information.

Before reporting a problem

  1. Confirm the installed version in Settings → Application.
  2. Restart UpTier and retry the exact action once.
  3. Note whether you launched an installed build, Microsoft Store test package, or Visual Studio/development build.
  4. If an import failed, record the file type, bank or broker name, column headings, and exact on-screen error. Do not upload the original file.
  5. Search the public tracker for the same version and symptom.

What to include

  • UpTier version and Windows version;
  • the page and action that failed;
  • the exact error text;
  • whether the issue repeats after a restart; and
  • a small, invented sample row when an import layout is involved.

Send privacy, security, and deletion matters to the private support mailbox. Do not place sensitive details in a public issue.

Local data and recovery

Installed and development builds use separate profiles. An unpackaged Visual Studio launch uses the development profile and does not display the installed app's data.

The direct Windows installer can preserve the encrypted Profile Vault during uninstall. Do not manually delete the UpTier application-data folder unless you intend to permanently remove all profiles and settings. Follow the Data Deletion instructions when that is your goal.

Support scope

UpTier is financial planning software, not a broker, bank, tax service, or investment adviser. For a calculation question, include only non-sensitive example inputs and the expected result so the formula can be reproduced safely.
